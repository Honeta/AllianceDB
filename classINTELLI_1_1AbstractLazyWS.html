Detailed Description

An abstraction of lazy window slider, also inherited by other lazy window slider.

Author
Tony Zeng
Note
detailed description: To init and run, follow the functions below to start a WS
  • Configure the window type, time or count, setTimeBased
  • Configure window length: setWindowLen
  • Configure slide length: setSlideLen (default is 1 if not called)
  • Set parallel executing behavior on SMP,setParallelSMP
  • Optional, (setRunTimeScheduling)
  • To make the parallel join processors started, initJoinProcessors
  • Feed tuples feedTupleS or feedTupleR
  • Terminate, by terminateJoinProcessors
